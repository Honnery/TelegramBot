#Small project for Diploma

## Preinstallation

Neo4j: Bloom + browser

##Installation

### Download project
```
git clone https://github.com/Honnery/TelegramBot.git
```

### Configure settings in files: 
 - configs/database.ini
 - configs/telegram.ini 

### Install requirements
```
pip install -r requirements.txt
```
